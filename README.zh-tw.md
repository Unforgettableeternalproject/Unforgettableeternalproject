# 嗨，大家好，我是 Bernie (U.E.P) 👋

我的目標是成為一名前端工程師，並不斷提升我的技能和知識。

希望這並不會是太過遠大的夢想。

## 技能

- **程式語言**: Python 🐍, C# ⚙️, JavaScript 🌐, Java ☕, C++ 💻
- **興趣**: 前端設計 🎨, 機器學習 🤖, 和人工智慧模型 🤖
- **一些有的沒的**: 我不知道，我自認還蠻會帶領別人的吧?
